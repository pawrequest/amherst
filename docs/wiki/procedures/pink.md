# Unpacking Hires
When a hire is returned, it is unpacked and checked to make sure that everything is present and correct.  

If the equipment is brought back in-person by anyone from the hiring organization, unpacking and inventory should be done in front of them, so that they can see what is missing - doing this helps to counter false denials of missing kit.

Returned Hires are marked in Commence as either:
- "Returned OK" if everything is returned un-damaged.
- "Returned with problems" if anything is missing, or is broken / damaged.

Once all equipment is returned or paid for the hire is marked as "Closed" in Commence.


## Dealing With Missing / Damaged Equipment
If anything is missing, try to phone the customer, or their organization, and tell them what is missing, and ask if they have it, or can look for it.
Or if they know it is “lost” and it makes sense for us to send an invoice.
