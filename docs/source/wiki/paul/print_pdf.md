# Resize and array A6 PDFs as booklets on A4

## Installation:

Run shortcut at "R:\paul_r\install_a6_label_printer"

## Uninstall:

Run shortcut at "R:\paul_r\remove_a6_label_printer"

## Usage:

- Right click on a PDF file
- Select "Print PDF 2x A6 on A4"
- the end

if the input file is landscape output will be strange
