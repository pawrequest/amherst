# VOIP Features

Yealink T41S Handset
![](/img/phones/yealink_phone.png) 


## Transfer active Call between extensions:

1)	Transfer button `(1)`
2)	Dial destination extension `(001 - 003)`



## Move call from one phone to another:

Place call on hold (2)

Park call `(4, 2)`

On new handset retrieve call `(*88#,#)`

## Join existing call - 'Barge In'

Dial `(*33 + extension)`
Eg, for giles to join chris’s call he will dial `(*33002)`
